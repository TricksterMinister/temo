# Интерактивный шоукейс "Один человек. Не агентство."

Этот проект представляет собой самодостаточную статическую веб-страницу (cinematic scroll) для презентации ИИ-интеграций Теймураза Бенидзе (Темо) Евгению Чичваркину.

## Как запустить локально

Так как страница использует библиотеки и внешние медиа, для корректной работы рекомендуется запускать ее через локальный веб-сервер:

### Способ 1: С помощью Python (самый быстрый)
Если у вас установлен Python, выполните в терминале в этой папке:
```bash
python3 -m http.server 8000
```
И откройте в браузере: `http://localhost:8000`

### Способ 2: С помощью Node.js (Vite / Live Server)
Вы можете запустить встроенный Vite сервер из корня репозитория:
```bash
npm run dev
```

## Как заполнить реальные ссылки (для Темо)

Все ссылки и плееры подключаются динамически через конфигурационный объект. 
Чтобы обновить контент, вам не нужно переписывать верстку. Откройте файл `js/main.js` и заполните объект `CONFIG` в самом верху файла:

```javascript
const CONFIG = {
  youtubeChannel: "ССЫЛКА_НА_ВАШ_КАНАЛ",
  youtubeVideoEmbed: "ССЫЛКА_НА_EMBED_РОЛИКА", // формат: https://www.youtube.com/embed/ID_ВИДЕО
  musicEmbed: "ССЫЛКА_ИЛИ_КОД_ВСТРАИВАНИЯ", 
  orbisLink: "ССЫЛКА_НА_ORBIS",
  iceInstinctLink: "ССЫЛКА_НА_КОНСЬЕРЖА",
  podcastEmbed: "ССЫЛКА_НА_ПОДКАСТ",
  telegramUsername: "ВАШ_ТЕЛЕГРАМ_БЕЗ_СОБАЧКИ",
  phoneNumber: "ВАШ_ТЕЛЕФОН"
};
```

### Размещение картинок:
- Картинки для галереи Digital Art (4-8 штук) положите в папку `assets/img/art/`.
- Скриншот для блока Ice & Instinct положите в папку `assets/img/ice-instinct/`.

## Как деплоить

### Вариант 1: Netlify Drop (без терминала)
1. Откройте [Netlify Drop](https://app.netlify.com/drop).
2. Перетащите папку `build/` в окно браузера.
3. Дождитесь окончания загрузки, перейдите в Site Settings и измените домен на удобный (например, `temo-ai-showcase.netlify.app`).

### Вариант 2: Vercel CLI
Если у вас установлен Vercel CLI, выполните в этой папке:
```bash
vercel
```
И следуйте инструкциям на экране.
